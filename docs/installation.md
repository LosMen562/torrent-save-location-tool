# Installation

## Stable release

To install TSLT, run this command in your terminal:

```sh
uv add Torrent_Save_Location_Tool
```

Or if you prefer to use `pip`:

```sh
pip install Torrent_Save_Location_Tool
```

## From source

The source files for TSLT can be downloaded from the [Github repo](https://github.com/LosMen562/t_s_l_t).

You can either clone the public repository:

```sh
git clone git://github.com/LosMen562/t_s_l_t
```

Or download the [tarball](https://github.com/LosMen562/t_s_l_t/tarball/master):

```sh
curl -OJL https://github.com/LosMen562/t_s_l_t/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd t_s_l_t
uv pip install .
```
